# 🚀 Task Automation with Python Scripts

## 📖 Overview

This Python project automates the process of moving `.jpg` files from a source folder to a destination folder. It helps organize files efficiently and reduces manual effort by automating repetitive tasks.

## ✨ Features

* 📂 Automatically detects `.jpg` files
* 🔄 Moves files from one folder to another
* 🛠️ Creates required folders automatically
* ⏱️ Saves time and reduces manual work
* 💡 Simple and beginner-friendly

## 🛠️ Technologies Used

* 🐍 Python
* 📁 OS Module
* 📦 Shutil Module

## 📂 Project Structure

```text
Task_Automation_with_Python_Scripts/
│
├── main.py
├── README.md
├── source_images/
└── moved_images/
```

## ⚙️ How It Works

1. 📁 Checks if the source and destination folders exist.
2. 🏗️ Creates the source folder if it is missing.
3. 🔍 Finds all `.jpg` files in the source folder.
4. 🚚 Moves the files to the destination folder.
5. ✅ Displays the names of the moved files.

## ▶️ How to Run

```bash
python main.py
```
